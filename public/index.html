KuaDashboard — Help & Reference

Action Buttons

Logs Abre el stream de logs en el panel terminal. Soporta múltiples pods en tabs separados con timestamps.
YAML Ver y editar el YAML del recurso. Usa “Apply” para aplicar cambios al clúster en vivo.
Delete Elimina el recurso de forma permanente. Requiere confirmación antes de proceder.
Restart Rolling restart del Deployment o StatefulSet sin downtime. Equivale a kubectl rollout restart.
Scale Cambia el número de réplicas. Ingresa 0 para escalar a cero el deployment.
Cordon Marca el nodo como no-schedulable. Los pods existentes siguen corriendo pero no se asignan nuevos.
Uncordon Reactiva el scheduling del nodo. El nodo vuelve a aceptar nuevos pods.
Drain Evictúa todos los pods del nodo para mantenimiento. Combina cordon + evict automáticamente.

Panel Terminal

Logs en vivo

Haz clic en Logs en cualquier pod para abrir su stream. Cada pod se abre en un tab separado.

Controles del terminal
  • Clear — limpia la salida
  • Wrap — activa el ajuste de línea
  • End — salta al final
  • Stop — detiene el stream
  • Pop-out — abre en nueva ventana
Resize

Arrastra el borde superior del panel para cambiar su altura.
