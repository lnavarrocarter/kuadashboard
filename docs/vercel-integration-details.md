# KuaDashboard Integration

KuaDashboard integra Vercel para administrar proyectos y despliegues junto con AWS, GCP y Kubernetes desde un unico panel.

## Incluye

- Lista de proyectos y estado de deployments.
- Vista de activity, aliases, DNS records, cron jobs, webhooks y checks.
- Operacion centralizada sin cambiar entre multiples consolas.

## Seguridad y cumplimiento

Usa OAuth de Vercel y emplea tokens solo para funciones habilitadas.

## Additional Information

KuaDashboard is a local-first operations dashboard focused on visibility and control across Vercel, AWS, GCP and Kubernetes. The integration is designed for read/operate workflows such as reviewing project health, deployments, domains, aliases, DNS records, cron jobs, webhooks and checks from one place. Authentication is handled via Vercel OAuth, and tokens are used only for enabled features. No data is sold or brokered. This integration helps teams reduce context switching, standardize cloud operations, and speed up incident response with a single operational surface.
