## 3. Open source nature

KuaDashboard is an open source project with source code that can be audited by third parties. This implies:

- Code transparency and change traceability.
- Independent security and quality review capability.
- Permissive licensing (MIT) for technical adoption and evaluation.
- Community collaboration and continuous software improvement.

## 4. Security, trust, and traceability

To reduce distribution and runtime risks, the project incorporates trust controls:

- Code signing for distributed binaries.
- macOS notarization for origin and integrity validation.
- Publication of versioned and verifiable release artifacts.
- Encrypted credential handling mechanisms in the application.

## 5. Functional scope for evaluation

KuaDashboard provides centralized infrastructure and operations administration, including:

- Kubernetes resource management (pods, deployments, services, ingress, storage, and nodes).
- AWS and GCP service integrations.
- Real-time logs, interactive shell, and integrated local terminal.
- Persistent port forwarding for services and pods.
